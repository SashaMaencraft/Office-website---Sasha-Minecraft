# Классы Безопасности

### DragonCorp: Классы объектов

**Класс Безопасный (Safe - DC)**

* Объекты, не представляющие непосредственной угрозы.

**Класс Опасный (Euclid - DC)**

* Объекты с непредсказуемыми свойствами, потенциально опасные для людей.

**Класс Таинственный (Keter - DC)**

* Объекты, представляющие наибольшую угрозу, свойства которых плохо изучены.

**Класс Аномальный (Anomalous - DC)**

* Объекты с необычными свойствами, не представляющие прямой угрозы.

**Класс Безвредный (Neutralized - DC)**

* Объекты, которые больше не представляют угрозы.
